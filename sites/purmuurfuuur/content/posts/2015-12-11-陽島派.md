# 陽島派

|  |
| --- |
|  |
| photo credit: [Pie Overview](http://www.flickr.com/photos/28399816@N00/300749794) via [photopin](http://photopin.com/) [(license)](https://creativecommons.org/licenses/by-nc/2.0/) |

高中家政課以外，人生的第一次廚藝教室體驗。  
  
因為吳太快手，竟然報名到離家近甜點教室的秒殺鹹派課，前些日子因緣際會下吃到那家店的鹹派，冷冷吃依然驚艷全場（說是全場但我家只有十坪而已，應該也沒有人在乎這件事情但硬要講一下），念念不忘的滋味竟然可以親手複刻怎能不心動。  
  
期待了一兩週，特地準時下班為了怕趕不上課程開始。同學們都準時到場，這大概是我近年來參加過難得可以準時開始的活動，不知道為什麼大家都很愛遲到，我想大概是因為日本帝國政府殖民的時間不夠改變基因台灣人都很隨和。  
  
總之進入平常走不進去的後場廚房，心情是激動的，有一種特別的尊榮感。（其實只要有報名課程就進得去了，可是要報名到也不簡單一次上課只有六個名額，訂位高手名不虛傳都要脫帽致敬一下）  
  
個頭嬌小的老師課程一開始先自我介紹並且介紹了助教一番，走一個熱情奔放又好相處的少女路線。  
  
瞬間就出現了一盤生南瓜，老師開始說這個很隨興地大家就是當自己奧力佛就對了，手不停地胡亂灑胡椒、蒜頭、以及因為種死所以乾脆把葉子風乾備用的香草（應該是迷迭香吧不過我竟然連第一步驟的細節都回想不起來果然魚腦），應該還有奶油喔，因為有特別介紹澄清奶油因為去除了固型物與雜質所以發煙點提高，比較不容易烤焦又可以提供奶油香。  
  
大蒜不用去皮只要稍微拍扁，大蒜皮可以保護大蒜仁避免太焦。是說有多怕烤焦，我覺得微焦大蒜也美賣呀。依照結果論，大蒜還因為沒有拍很扁又沒去皮，加上數量不多，所以最後的南瓜沒什麼大蒜味。據說也可以使用九層塔（筆記）。  
  
怕乾的老師一直講話把場子搞的濕濕的，大概是走一個怕冷場的嗨咖路線，所以只要停止講話想遞麥出去發現沒人接以後臉上就隱隱浮現不安感。但央大的名言「會讓場子乾的人本身也不介意場子乾」可是所言不虛，希望老師未來的人生路可以咀嚼到這件體會，少一點熱場強迫症，多一點料理的科學。  
  
接著端出了預先炒好一半份量的肉燥口味焦糖洋蔥，以及切功驚人的洋蔥絲（結果是刨絲器在厲害），身為吃貨，替同學爭取試吃是一定要的（根本就是因為晚餐沒吃飽），獲得一口洋蔥，阿，我離端回鹹派又近了一步呀！  
  
因為炒洋蔥這件事情沒什麼技術成份，只要努力就有成果，所以說明完以後就交由助教炒到世界末日（世界末日梗老師特愛，貫穿全場，我都想要聊一下日和的浦島太郎梗了）。  
  
接著在等世界末日洋蔥變焦糖與南瓜烤熟的空檔，我們開始了一個揉派皮的動作（贅字很多我在練習當腦殘記者嗎）。首先介紹材料，麵粉本體採用折衷的高筋低筋一比一配方，把奶油切成小丁狀，方便揉入麵粉，這時候冰手體虛人簡直有天生優勢，奶油都不會快速融化真是好生羨慕，一個快手才有好塔皮的概念，混成鬆散的沙狀手感以後就可以做出一堵圓形粉牆。  
  
奶油融在麵粉裡如果顆粒很均勻，成品派皮的蓬鬆感來源－空氣細縫就會很均勻分布，要揉得很均勻又不會因為揉得太久結果出筋派皮硬，是有一定難度的。  
  
搞定奶油沙後，在粉牆裡倒入糖鹽混和物以及黃色的蛋黃冰水預拌液後，努力地把手中的沙粉拌拌拌揉成團狀，成團以後要讓麵糰放鬆一下。一般在家裡放冷藏放個幾小時（是說到底要放多久我也忘記了），不過在教室時間不夠所以包在保鮮膜裡面丟去冷凍冰（中間還要去翻面才冰得均勻）。放鬆的作用是什麼老師好像沒有講，但我想應該是查得到而且我跟圖書館預約了料理的科學這本書，到時候裡面也會說，哼哼。（是在有恃無恐什麼，而且也沒人在意這個）  
  
疑，揉完麵糰炒完洋蔥處理完南瓜以後的空檔在做什麼，我有點忘記了，不過好像是因為太乾了所以老師開始張羅要招待我們的甜點，我超期待可以吃到一整顆栗子栗子栗子或者沒賣完的盪高，結果是我想太多，老師準備了［擠一坨鮮奶油胡蘿蔔蛋糕］與［咬一口莓果布朗尼］配［甜在心紅心芭樂茶］。（到底是為什麼要突然走時常在這裡的命名路線）  
  
胡蘿蔔蛋糕的肉桂味很香，鮮奶油也是好的，但就是沒有胡蘿蔔味。布朗尼微波以後熱熱的吃配上剛擺上去的冷凍莓果就是一個冰火九重天加上酸甜二重奏的概念，作為招待的甜點我給它九十分。還有紅心芭樂味濃且甜度適中的飲品，由平常光顧時都會在店裡的晚娘阿姨端入場，這個中場休息我是很滿意的。  
  
中場邊吃邊聊邊查看烤南瓜進度，此時放置 play 的洋蔥世界末日也到來，冰箱裡的派皮也可以拿出來成型到烤模了，一種左右逢源的錯覺。  
  
本場教學重點來到－灑手粉，感覺好威風呀，灑得一桌超不均勻然後左手抹開的過程，都想唱黑色幽默了。把派皮擀開成圓形，戳出一些不穿破塔皮的凹洞，放到烤模裡，壓邊邊兒，率性地用刀背削掉多的派皮。派皮最好可以均勻厚度，烤的時候收縮程度才會一致，而且派邊同樣高度的話，裡頭可以填的料才多呢（撥瀏海）。壓好的派皮，填入塔石以後就可以送入烤箱，塔石就是小小顆有重量的填充物，放在塔皮上，填滿預計要裝料的空間，目的是要協助塔皮成形，底面不踫龜，其實只要有重量而且可以把空隙填好然後又耐烤的東西就行，這次是用烤焙紙墊著然後撲滿看起來已經烤很多次的綜合豆。  
  
等待烤皮的時候，空檔比較多一點，所以有個 BONUS 教學，捏甜派的塔皮，此時獲得一個塔皮分著吃，好好吃，甚至覺得比填了檸檬餡一起吃還好吃。甜派的想當然比較難捏，因為烤模是沒有底的，先在烤圈上抹哈味發酵奶油（最喜歡在場都沒有人知道只有陽島家兩人自己樂得笑呵呵的無聊梗）協助塔皮貼緊烤模以形成底緣尖挺的直角，每個人有三片的練習機會，基本上就是快手壓壓捏捏轉轉轉，老師說大家的都滿利害都可以去打工了，頭腦簡單的我整個被稱讚到捏得起勁還認真思考一下我下班以後幾點可以去打工的這事兒，被捧得舒心。  
  
適當地讓學員獲得成就感真的是教學的重點呀！舒心時間結束後，心情愉快地著手進行蛋奶液，蛋奶液就是，有哈味奶油、非林鳳營牛奶與鮮奶油混和的神秘液體、蛋黃超緊實的一顆全蛋加上一顆蛋黃、黑胡椒、香料、起司粉，打打打勻，加鹽，很快就完成。  
  
然後又進入怕乾老師恐慌症時間，因為東西做好了塔皮還沒好，本來想帶動唱可是老師唱的歌我只有聽過不會唱，噢我好傷心，本來還想說可以大家唱一下應景的叮叮噹沒想到老師唱的是叮噹（明明就是梁靜茹），是說老師果然是少女心跟我這種真的要唱歌詞的話 21 世紀的歌只唱的出來林俊傑或周杰倫的人不一樣（倒）。因為太乾所以老師展示了她們新換的小門可以上下拉開露出百葉還有紗窗，並且看到外頭的江惠貞看板 say hi。  
  
乾時間也沒很久（大概只有老師以為過了一萬年吧竟然還邀大伙去跟江惠貞打招呼），我們的塔皮已經可以填料了，拿出來以後要先去除塔石，本來老師跟助教合作無間地清掉了一個塔，可是我很機歪提出「平常一個人可以完成這件工作嗎」的問題，老師為了示範一個人也可以很快樂於是灑了六分之一桌的黃豆。  
  
撿好豆子後就可以替烤好的半成品塔皮加入愛的肉燥洋蔥與烤南瓜，充滿期待地鋪滿滿，加上起司，填滿蛋奶液，接著送入烤箱烤，後面我們做了什麼我忘記了，約莫就是一般社會化的人都要會一點的無意義聊天殺時間，好像有說明一下講義的內容，可是講義寫得好精簡我覺得這篇流水帳還比較有參考價值，只記得漬番茄配蔥抓餅很好吃這件事，而且我問了配漬番茄的蔥抓餅可不可以加蛋的問題事後回想起來好像沒有得到確切回覆，撐到烤得差不多了就拿出來鋪比薩起司，再送回烤箱最後一波火力放送。  
  
愉快的時間很快就結束了，帶著香奔奔的兩個鹹派回家，順道買了份雞排。充滿歡快的一個晚上，課程規畫得很有心，有努力把無聊的苦力減少然後想活動填滿烘焙課惱人的等待時間，留下最有趣最有成就感的手做部分給學員享受。如果要再上一次課，很是願意參加。  
  
我有點不清楚澄清奶油的製作教學是安插在哪一個空檔中間殺時間，不過這種一邊聊天一邊學到一些生活中的小事，記憶很深刻。（澄清奶油就是把奶油加熱以後上面浮起的白泡撈掉然後下頭沉澱的固化物濾掉，留下中間清澈的油，可以提高發煙點，也可以當成無水奶油使用，冰起來的質地比未處理過的奶油扎實）
