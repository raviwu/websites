# 舌尖天份

|  |
| --- |
|  |
| 豬肉紙舌 |

  
在網路上看到「超級味覺者」（[Supertaster](http://www.bbc.co.uk/science/0/22941835)）的資訊，興奮地在吃中餐的時候吐出舌頭想要得到「喔我的味覺其實挺靈敏的嘛只是欠訓練」的結論。  
  
這世界總是事與願違，我的舌頭平到不行，乾淨程度簡直清新，就像一張純潔的豬肉紙一樣，滑順地粉紅。想著回家看一下吳太的舌頭是否天份比我好一些，腦中被各式雜務塞滿然後就擱置一旁。  
  
昨晚飯後閒聊中討論到有關味覺的話題，腦袋像被雷打中一樣突然想到要驗證這件事，興奮地用閃光燈拍下吳太的美舌。  
  
充滿紅咚咚的味蕾呀！又大又密又洋溢的味蕾呀！難怪總是可以說出「感受在口中化開的後味」這種有深度的品嚐技巧，我總是有聽沒有體會，原本只覺得一定是自己不夠用心，但真相揭曉，原來是因為我根本沒有分佈那麼多味蕾去體會。  
  
老天爺是公平的，每個人的天份樹，真的都長不一樣。往光明面思考，至少我有一個忠誠的外包舌頭可以嚐出我用心腦補就能想像的味道，豈不也幸福無比。
