# 第一次車遊驕傲月

一直都不是會去參加遊行的人，第一次的參與是婚姻平權那次，好像沒有遊行很遠，從台大醫院走到景福門附近就開始塞住，大部分時間是和坐在凱道上硬硬的地板，拿了一些貼紙跟傳單。

|  |
| --- |
|  |
|  |

那張粉紅色的傳單被我拿去貼到公司的布告欄上，好像獲得了一些同事的出聲支援，在歐美人比例高的環境裡好像有很被包容的感覺。那幾張貼紙忘記去處了，大概率是貼在那年的日記本上。寫了十幾年的日記，從流水帳跟無意義的工作抱怨呻吟，後來努力往生活上多點感受累積。寫的人早已忘記自己在寫什麼字，看的人偶爾隨機翻頁會遇到可能有好笑的內容。

幾年後同婚法通過，我們從迪士尼慶生完後登記結婚，那一天我們吃了詹記麻辣鍋，穿了顏色有搭配的情侶裝。換了身分證，配偶欄上註記了彼此。

第一次的同志大遊行是在2025年要去麵包祭找朋友吃吃喝喝的計程車上，我們預留了晚上要另外帶給另一個朋友的伴手，帶著萬全準備的保冷袋，甚至被說是箱子。車上的我們有點靜默，前一晚我的不當發言還讓她心裡很不舒服，當天下午也沒能好好地讓氣氛變好。

司機因為遊行封路感覺不耐煩，我們從信義轉光復南路沿路上看到很多滿臉笑容的人，我心裡想著，是大遊行呢，能這樣參與真好。我過著很幸福的生活，我身邊的她跟我一起努力著要繼續好好攜手，後來好一些能夠多說些話的時候，他說她那天想著這是我們的天結果我們在吵架。

彼此相知相惜，遠比相愛要有更多更多的自我揭露，我帶著自己的問題到關係裡。我記得妳教我要怎麼任性，其實我一直對妳很任性。妳不是謝謝教的，我的感謝應該要長成完整的自己、有韌性的自己、可以跟妳攜手坦率面對跟克服挑戰的自己。
