# 小事累積的安心感

|  |
| --- |
|  |
| photo credit: [feathers](http://www.flickr.com/photos/72429059@N00/3408251208) via [photopin](http://photopin.com/) [(license)](https://creativecommons.org/licenses/by-nc-sa/2.0/) |

  
睡到一半醒了，臥室異常地亮，車聲微微，時間還在昏暗。  
  
在書桌前忙著什麼的另一半聽到咿咿聲響，連忙放下手邊的事問著：「怎麼了？」  
  
迷迷糊糊地我也問著：「咿咿怎麼了？」  
  
「眼睛好像進東西了在查看。」  
  
「有出來了嗎？」  
  
「恩還在弄呢，妳快睡呀。」  
  
試著闔上眼努力回到睡眠的我，過沒幾分鐘還是睡不著，咿咿不絕地要人哄睡。  
  
回到床上的太太湊到身邊溫柔地摸著我的頭。  
  
「趕快睡喔，才四點多明天會沒精神工作。」  
  
突然一陣安心感地輕聲嘻嘻笑：「好愛妳喔。」  
  
「我也是呀！」  
  
為了愛再努力一下吧，閉緊著眼，沉沉睡去。
