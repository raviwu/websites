# 愛心娘被

|  |
| --- |
|  |
| photo credit: [BIG HUG!](http://www.flickr.com/photos/98413464@N03/26637853142) via [photopin](http://photopin.com/) [(license)](https://creativecommons.org/licenses/by-nc/2.0/) |

  
因為本人的特殊渲染力，被我長期臨幸過的被子們，紛紛充滿米味。不管是春夏的涼被，亦或是冬日大棉被。曬了幾日依然走不出米味繞樑，最終紛紛宣判棉死。  
  
家裡的採購大臣——屈米央氏，為了即將來到的夏天，焚膏繼晷、日以繼夜地張羅新一季的米味受害者守護者，殷殷期盼又突如其來的包裹收到，裡面是滿載愛意的娘被。  
  
天絲的涼感，源於滿溢的期盼。  
  
卑職米官獲賜一席又滑又涼的新被，得名「若竺」，夾帶虛若懷竹之勉勵，使米如沐春風。  
  
央氏給自己添了一縷以純潔為底，藍天綴邊，金枝妝點的「天使」，輕柔披覆深情萬千。  
  
今夏的「若天竺使」，除了將凡俗熱氣昇華之外，還象徵著堅定信念終將克勉世事艱難的取經精神，為陽島家展開令人期待的新紀元。感恩央，讚嘆央。
