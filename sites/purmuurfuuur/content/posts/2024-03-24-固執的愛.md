# 固執的愛

|  |
| --- |
|  |
| 熱熱的大皇宮看玉佛 |

  

曬了整下午的兩人

在 Iconsiam 的手標泰奶店 她下令要我去旁邊坐

讓她排隊就好 她說昨天有買過一次了她滿懷信心知道怎麼點餐怎麼買

我坐著遠遠觀察她剛說的待會要去逛的類 OMAKE 店是不是新的伊甸園 渾然不知已經買好奶茶還喚了我幾聲

終於也是一陣疲倦的她坐下以後第一件事 是來替我刮痧

後來我領著她一起走進思思念念麻衣店 她挑了四五件 都是給我的

她對我的固執的愛 讓我賴以存在 😌
